## Installation workflow

For a correct wall installation, follow the steps below **in the specified order**:

1. **Prepare the VELUX® wall remote control**  
   See the [remote control](remote-control.md) section.

2. **Connect to the power line and install the device in the wall**  
   See the [wall installation](wall-installation.md) section.

3. **WIFI Setup**  
   See the [WIFI](wifi.md) section.

4. **Set up Home Assistant integration**  
   See the [Home Assistant](homeassistant.md) section.
