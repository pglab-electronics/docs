# E-VLXESP32

The **E-VLXESP32** is a compact electronics module featuring an integrated **temperature and humidity sensor**.
  
It is designed to interface with battery-powered **VELUX®** wall remote controls **(*)** (models **KLI311, KLI312, KLI313**) and allows users to explore and test smart automation for remotely controlling **VELUX®** motorized skylight windows.

The E-VLXESP32 connects seamlessly to your Home Assistant system over WiFi. Powered by ESPHome, it automatically detects all sensors and features, letting you create dashboards and automations in just minutes—without any complicated setup.

## Product Versions

The E-VLXESP32 is designed to adapt to different installation needs and is available in two distinct versions: the **Standard Version** and the **KIT Version**.

| Feature | Standard Version | KIT Version |
| :--- | :--- | :--- |
| **Target Use** | Permanent wall installation | Makers and DIY enthusiasts |
| **Enclosure** | Pre-assembled, professional housing | 3D printable (DIY) |
| **Power Supply** | **Mains line power** (Direct AC wiring) | **USB cable only** (5V DC) |

### Standard Version
The Standard version is a fully assembled, plug-and-play device intended for **permanent wall-mounted installation**. It connects directly to your home's **mains electrical line**, eliminating the need for external power bricks or visible cables. It is the ideal choice for a clean, professional, and integrated smart home look.

### KIT Version
The KIT version is tailored for **makers** who prefer a hands-on approach. This version comes as a bare PCB, allowing you to **3D print the enclosure yourself** using our official designs. To simplify safe DIY handling and desktop placement, the KIT version is **powered exclusively via a USB cable**, requiring no high-voltage AC wiring.
