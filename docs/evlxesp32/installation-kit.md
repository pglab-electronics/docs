## Installation workflow

For a correct installation, follow the steps below **in the specified order**:

1. **Prepare the VELUX® wall remote control**  
   See the [remote control](remote-control.md) section.

2. **DIY 3D Print the enclosure**
   See the [3d printing](3d-printing.md) section.

3. **Connect to USB-C cable and power E-VLXESP32**  

4. **WIFI Setup**  
   See the [WIFI](wifi.md) section.

5. **Set up Home Assistant integration**  
   See the [Home Assistant](homeassistant.md) section.
